# Digital-Agency-website
Digital Agency website interface concept using Figma AI, focusing on clean layout, structured content, and intuitive user flow.
Design sections included:
 • Introduction / Hero section
 • Project showcase
 • Client testimonials
 • Footer layout

To enhance efficiency and design quality, I utilized Figma AI features such as:
 • Text & Content Intelligence Tools
 • Image & Visual AI Tools
 • Layer Management & Organization
 • Prototyping & Interaction Intelligence
 • Search & Asset Discovery

I also used plugins including:
 • Iconify
 • Brandfetch
 • Unsplash
which maintains visual consistency and branding support.
